## 👤 About Me

> **"미래사회와 직업세계를 연구하며, 청년·성인·베이비부머 퇴직자까지 모든 세대의 커리어 경력설계와 전직지원을 돕고 있습니다. IT 분야 20년 경력을 바탕으로 교육정보화와 커리어 컨설팅을 융합한 실전형 전문가입니다."**

안녕하세요, **직업미래연구소 소장 정동엽**입니다. 🎓

직업학 연구의 깊이와 **IT 20년 현장 경험**, 미래학 전문성을 결합하여 **미래직업·진로·커리어**를 전략적으로 설계합니다. 청소년부터 베이비부머 퇴직자까지 모든 세대가 변화하는 직업세계를 준비할 수 있도록 함께합니다.

- 🔭 지금은 **미래사회와 직업세계 연구 · 커리어 경력설계 · 전직지원**에 집중하고 있습니다
- 🌱 **생성형 AI 시대의 AI 리터러시**와 디지털 기반 커리어 교육을 연구합니다
- 💬 진로지도, 창업·창직 컨설팅, 직업심리 상담이 전문 분야입니다
- 🏫 **동아방송예술대학교 겸임교수**로 창의융합교양학부에서 강의하고 있습니다
